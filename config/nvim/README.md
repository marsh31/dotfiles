# Neovim Config

marsh neovim config.  


#### フォルダ構成

```txt
~/.config/nvim
|-- init.lua
`-- lua
    |-- config
    |   |-- options.lua
    |   |-- keymap.lua
    |   `-- autocmds.lua
    `-- plugins
        |-- init.lua
        |-- lsp.lua
        |-- completion.lua
        |-- formatters.lua
        |-- kms.lua
        `-- others.lua
```


#### Plugin List

プラグイン一覧。  

| フラグ              | 状態説明                                                     |
| ------------------- | ------------------------------------------------------------ |
| (使用検討):         | まだ未使用だけど使用を検討している。                         |
| (使用):             | 使用中。                                                     |
| (コンフィグ見直し): | 使用しているけどコンフィグを見直しする。                     |
| (廃止検討):         | まだ使用しているけど、廃止、別プラグインへの移行を検討する。 |
| (廃止):             | もう廃止。使わない                                           |
